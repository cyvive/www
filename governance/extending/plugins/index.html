    Extending with custom Plugins¶

    Warning

    This functionality was originally offered by the parent company as a separate product Fathomable. It is undergoing consolidation into Cyvive's core and is restricted to beta customers at this time. Please contact your account manager via live-chat should you wish to participate.

    Summary

    integrating with the core Cyvive.io functionalities for custom gain.

    Architecturally Cyvive.io provides a series of custom events and actions that can be called as part of the core functionality. These are supported endpoints and are stable datastrutures that can be hooked into for any plugin development.

    These endpoints can also handle middleware, should the need arise.

    Events¶

    core events emitted regularly are as follows:

    • generated:app
    • generated:group
    • generated:namespace
    • updated:github
    • updated:container
    • configSync:check
    • configSync:update
    • isReady
    • regenerate:app
    • regenerate:group
    • regenerate:ns
    • refresh:generated
    • refresh:db
    • initDB
